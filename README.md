# 🎓 Student Management System

A simple **Java Swing + MySQL** based desktop application to manage student records, attendance, and logs efficiently.

---

## 📌 Features

- 🔐 Login System (Admin access)
- 👨‍🎓 Add / Update / Delete Student Details
- 📊 Mark and manage Attendance
- 📄 View Student Records in Table (JTable)
- 📜 Audit Log tracking system changes
- 🔄 Refresh and real-time database updates

---

## 🛠️ Technologies Used

- Java (Swing - GUI)
- MySQL (Database)
- JDBC (Database Connectivity)
- NetBeans / VS Code
